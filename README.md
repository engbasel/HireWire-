**HireWire** هو بوت ذكاء اصطناعي مستقل يقوم بمسح وتحليل مشاريع البرمجة والتطوير عبر 4 منصات عمل حر كبرى. يقوم البوت بتصفية العملاء غير الجديين، وتحليل متطلبات المشروع باستخدام **Google Gemini**، ثم يرسل لك أفضل المشاريع التي تناسب مهاراتك مباشرة على **تليجرام**.

---

## ✨ المميزات الرئيسية

- **Web Dashboard**: واجهة مستخدم احترافية للتحكم في البوت (بدء/إيقاف)، تعديل الإعدادات، ومشاهدة السجلات (Logs) في وقتها الفعلي.
- **تغطية شاملة**: يدعم (مستقل - نافذلي - PeoplePerHour - Guru).
- **تحليل ذكي (AI)**: تقييم كل مشروع بواسطة Gemini 3.1 لرؤية مدى توافقه مع خبرتك.
- **تصفية الجودة**: استبعاد العملاء ذوي معدل التوظيف 0% للتركيز على الفرص الجدية.
- **تنبيهات فورية**: تصلك التقارير على تليجرام لحظة العثور على فرصة مطابقة.

## 🚀 التثبيت والتشغيل (Quick Start)

### 1. المتطلبات
- **Python 3.10+** مثبت على جهازك.

### 2. إعداد البيئة
افتح التيرمينال في مجلد المشروع وقم بتشغيل الأوامر التالية:

```bash
# إنشاء البيئة الافتراضية
python -m venv venv

# تفعيل البيئة (Windows)
.\venv\Scripts\activate

# تثبيت المتطلبات
pip install -r requirements.txt

# تثبيت متصفحات Playwright
playwright install chromium
```

### 3. تشغيل سرفر الواجهة (Dashboard)
بدلاً من تشغيل الكود مباشرة، سنقوم بتشغيل سرفر الواجهة الرسومية:
```bash
python api_server.py
```
بعد التشغيل، افتح المتصفح وادخل على الرابط التالي:
👉 **[http://localhost:5000](http://localhost:5000)**

## ⚙️ الإعدادات (Settings)

من خلال الواجهة الرسومية (Settings)، ستحتاج لإدخال البيانات التالية:

1.  **Gemini API Key**: من خلال [Google AI Studio](https://aistudio.google.com/app/apikey).
2.  **Telegram Bot Token**: من خلال البوت [@BotFather](https://t.me/BotFather).
3.  **Telegram Chat ID**: من خلال البوت [@userinfobot](https://t.me/userinfobot).

> **ملاحظة**: الواجهة توفر شرحًا تفصيليًا لكل خطوة داخل صفحة الإعدادات لمساعدتك إذا لم تكن لديك خبرة برمجية.

## 📂 هيكل المشروع (Architecture)

- `api_server.py`: سرفر Flask المسؤول عن الواجهة الرسومية.
- `main.py`: المحرك الأساسي (Orchestrator) الذي يدير عمليات البحث والجدولة.
- `scraper.py`: محركات البحث الخاصة بكل منصة باستخدام Playwright.
- `ai_agent.py`: دمج ذكاء Gemini لتحليل المشاريع.
- `dashboard/`: ملفات الواجهة الرسومية (HTML, CSS, JS).
- `database.py`: قاعدة بيانات SQLite بسيطة لمنع تكرار التنبيهات.

---

## 📜 السجلات (Logging)
يمكنك متابعة ما يفعله البوت في كل ثانية من خلال صفحة **Live Logs** في الواجهة الرسومية، كما يتم حفظ السجلات في مجلد `logs/` للرجوع إليها لاحقًا.

## 🤝 المساهمة
يتم تحديث البوت بشكل دوري لمواكبة تغييرات واجهات منصات العمل الحر. إذا واجهت أي مشكلة في سحب البيانات من منصة معينة، يرجى تحديث `selectors` في ملف `scraper.py`.
